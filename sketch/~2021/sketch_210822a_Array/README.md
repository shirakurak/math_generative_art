横方向に3分割しており、それぞれ別種の白黒グラデーションが描かれる。
`0`から`1`の連続的な値の変化を表現するために、`cos`を利用。
`0`から`PI`までに対する`cos`の値の絶対値を使用する。
横の長さと対応付けるため、`map`関数を利用する。
